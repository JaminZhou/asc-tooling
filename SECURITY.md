# Security Policy

If you believe you have found a security issue in `asc_tooling`, please report
it privately by email to `me@jaminzhou.com`.

Please do not open a public GitHub issue for:

- exposed App Store Connect API keys
- leaked `.p8` files or local secret material
- exported browser session cookies
- vulnerabilities that could expose account or release data

When reporting an issue, include:

- a short description of the problem
- reproduction steps or a minimal proof of concept
- impact details
- the affected command or file, if known

I will do my best to acknowledge the report promptly and follow up with a fix
or mitigation plan.
